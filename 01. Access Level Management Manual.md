## 📌 System Overview

The SC103 Proximity Card RFID Time Attendance System and Access Control System with USB is a specialized device designed to provide secure access control and track employee attendance using RFID technology. Below is an overview of its description, components, usage, and working principles.

---

## 📖 Description

The SC103 Proximity Card RFID Time Attendance System integrates both access control and time attendance functionalities into a single system. It is designed to enhance security and efficiency in workplaces, educational institutions, or any environment where controlling access to premises and monitoring employee attendance is necessary. This system uses Radio Frequency Identification (RFID) technology to read proximity cards that allow authorized personnel to gain access. It also tracks the attendance of employees by recording the time they enter or exit a designated area.

The system features USB connectivity, which simplifies the process of transferring data (such as attendance logs) between the device and a computer for analysis and record-keeping.

---

## 🧩 Components

### 1. Proximity Card:

* These cards contain embedded chips that store unique identification numbers. The cards do not require direct contact with the reader to function.

### 2. RFID Reader:

* The reader detects and reads the data from the proximity card when it is held near the reader. It communicates with the control unit to allow or deny access.

### 3. Control Unit:

* This is the central processing unit that stores data, such as access logs and attendance records. It also controls access based on the information received from the RFID reader.

### 4. Access Control Door Lock:

* The system is integrated with an electronic lock on doors, gates, or turnstiles. When the reader verifies the user's credentials, the lock is released to allow access.

### 5. USB Interface:

* This component allows data transfer between the system and a computer for backup, further analysis, or reporting.

### 6. Software:

* The software is installed on a computer to manage and analyze attendance records, generate reports, and monitor access control settings. It provides an interface to monitor logs, adjust settings, and manage users.

---

## 🎯 What is it used for?

* **Access Control:** It restricts access to specific areas based on user credentials, ensuring that only authorized personnel can enter certain areas.
* **Time Attendance:** The system records the times when users clock in or out, automating attendance tracking. This is particularly useful for companies and organizations to manage employee hours, payroll, and attendance records.
* **Security:** By controlling access to sensitive areas, it ensures physical security, preventing unauthorized personnel from entering restricted zones.
* **Data Management:** The system helps maintain digital logs of access events, such as entry and exit times, for reporting or auditing purposes.

---

## ⚙️ How it works

### 1. User Authentication:

* A person presents their RFID proximity card to the reader. The card is scanned by the RFID reader, which transmits the card’s unique ID number to the system.

### 2. Data Verification:

* The control unit compares the scanned ID with the database to verify if the person is authorized for access. If they are authorized, the system triggers the door lock mechanism to open and allows the individual to enter.

### 3. Attendance Tracking:

* The time of entry (and possibly exit) is recorded by the system to track attendance. The data is stored in the system’s memory or transferred to a connected computer via USB.

### 4. Data Transfer:

* Attendance data, access logs, and other relevant information can be transferred from the control unit to a computer via USB, where it can be processed, analyzed, or printed as reports.

## 🪪 Register a New Card User

### 4. Register a New Card User:

To register a new user, follow these steps:

---

### 🧾 Step 1: Retrieve the Card Number:

1. Tap the RFID proximity card against the physical device.

The system will display the Card Number. For example, 2867140459 will be provided as the card number.

---

### 🧾 Step 2: Add the Card Number to ZKTeco Software:

1. From the dashboard, navigate to the User Management section.

   * The User Management Dashboard will be displayed.
2. To register a new user, click on the "+" (Add User) icon.

---

### 🧾 Step 3: Enter User Details:

Fill in the user information as follows:

* **Name:** Enter the full name of the user.
* **ID:** Enter the user's identification number.
* **Card Number:** Enter the card number obtained from the physical device (e.g., 2867140459).
* **Gender:** Select the user's gender.
* **Job Title:** Enter the user’s job title.

Once all details are entered, click on the **"Right"** option to proceed to the next step.

---

## 📤 User Registration Completion

The system will successfully register the user in the software. Once the user is enrolled, the data needs to be uploaded to the device.

---

### 📌 Step 1: Upload User to Device:

1. Click on **"From PC to Device"** to initiate the upload process.
2. Select the user you wish to upload to the ZKTeco device.

---

### 🚪 Step 2: Assign Access to Entry and Exit:

1. To ensure the user is granted access to the correct doors, you must assign the user to both the Check-in (CHK In) and Check-out (CHK Out) doors.
2. Select the appropriate access points (e.g., Ground Floor Entry Door of the Hyatt Office).
3. Click **Upload** to transfer the user data to the selected devices.

---

## ✔️ User Management Finalization

Once the user has been successfully uploaded to the devices, close the User Management Dashboard.

* Confirm by pressing **Yes** when prompted to exit the system.

## 🔍 Verification of User Access

To ensure that the user has been successfully registered and granted access:

1. Test the RFID card at the Ground Floor Entry Door.
2. Verify that the user can successfully access the door for both Entry and Exit operations.

If the card number and the user credentials are correct, the door should open for the registered user.
